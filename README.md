Download & Ausführung der Anwendung
Die Anwendung kann über Code → Download ZIP heruntergeladen werden.

Nach dem Download den Ordner entpacken und die Datei
LernkartenAnwendung → LernkartenApp → LernkartenApp.exe
als Administrator ausführen.

Sollte beim Start eine Sicherheitswarnung erscheinen, kann diese über „Weitere Informationen“ → „Trotzdem ausführen“ (unterstrichener Text) umgangen werden.

Die Warnung wird angezeigt, da der Herausgeber als unbekannt gilt. Die Anwendung enthält keine schädlichen Dateien.

Ein Beispieldatensatz (Datei1) kann in der Anwendung geladen werden, um sämtliche Funktionen zu testen.
